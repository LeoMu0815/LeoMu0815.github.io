Typora 快捷键

文件

  • 新建：Ctrl + N
  • 新建窗口：Ctrl + Shift + N
  • 打开：Ctrl + O
  • 快速打开：Ctrl + P
  • 保存：Ctrl + S
  • 另存为：Ctrl + Shift + S
  • 偏好：Ctrl + ,
  • 关闭：Ctrl + W

视图

  • 显示隐藏侧边栏：Ctrl + Shift + L
  • 源代码模式：Ctrl + /
  • 专注模式：F8
  • 打字机模式：F9
  • 切换全屏：F11
  • 实际大小：Ctrl + Shift + 0
  • 放大：Ctrl + Shift + =
  • 缩小：Ctrl + Shift + -
  • 应用内窗口切换：Ctrl + Tab
  • 打开DevTools：Shift + F12

段落

  • 标题：Ctrl + 1/2/3/4/5
  • 段落：Ctrl + 0
  • 增大标题级别：Ctrl + =
  • 减少标题级别：Ctrl + -
  • 有序列表：Ctrl + Shift + [
  • 无序列表：Ctrl + Shift + ]
  • 任务列表：- + 空 + [ + 空 + ] + 空
  • 增加缩进：Ctrl + ]
  • 减少缩进：Ctrl + [

引用

  • 目录：[ + TOC + ] + Enter

无法在Hexo博客上显示

  • 软换行：Shift + Enter
    硬换行：空 + 空 + Shift + Enter
    换段：Enter

  • 代码：Ctrl + Shift + `

  • 脚注：[^ 1 ]

  • 插入链接：Ctrl + k

  • 锚点：[ + 内容 + ] + ( + # + 标题名 + )

引用：Ctrl + Shift + q

$$
数学符号：Ctrl + Shift + M
$$

制表
Ctrl + T
  • 插入图片：直接拖动到指定位置即可或者Ctrl + Shift + i
  • 分割线：：— + 回车

编辑

  • 粘贴为纯文本：Ctrl + Shift + V
  • 加粗：Ctrl + BU
  • 倾斜：Ctrl + i
  • 下划线：Ctrl + U
  • 删除线：Alt + Shift + 5

检索

  • 选中一整行：Ctrl + l

  • 选中单词：Ctrl + d

  • 选中相同格式的文字：Ctrl + e

  • 跳转到文章开头：Ctrl + home

  • 跳转到文章结尾：Ctrl + end

  • 搜索：Ctrl + f

  • 替换：Ctrl + h

[^ 1 ]: [ + ^ + 空 + 数字 + 空 + ]
