Hexo博客本地搭建
发表于 2020-07-29| 更新于 2020-07-29
阅读量:

要想快速搭建一个博客，可以使用现有的博客框架。
在这里我们使用的是hexo，一个基于Nodejs、快速简洁高效的博客框架。

环境搭建

安装Node.js

  • 官网传送门

选择LTS下载并安装。

安装过程一直默认下一步就行。

安装Git

hexo依赖Node.js和Git，因此还需要安装Git。

  • 官网传送门

由于“墙”的原因，Git在官网上下载会非常慢，这里提供更快速的下载。
安装过程一直默认下一步就行。

安装cnpm

在上一步中，我们已经安装好了Git，打开C盘右键可以看到Git Bash选项。
点击Git Bash打开界面，输入以下命令：
npm install -g cnpm --registry=https://registry.npm.taobao.org
等待安装完成后可以输入cnpm -v查看版本。

安装hexo

安装好了cnpm后，便可以利用它下载并安装hexo了。
在命令行中输入：cnpm install -g hexo-cli
自动下载安装完hexo后，同样可以输入hexo -v查看版本。

基本操作

博客初始化

经过以上步骤，所需的环境都已经准备好了。

  1. 选择一个你想要存放博客的位置，如C:\Users\Administrator

  2. 新建一个文件夹（取名随意，如blog即可，下称blog文件夹）用以存放hexo博客（如果后面不喜欢了可以删掉重来）

  3. 进入blog文件夹，右键打开Git Bash（要安装Git后才有）

  4. 输入hexo init命令初始化blog文件夹

等待一会（实际上要很久），便可以在blog目录下实现博客的初始化了。
这是已经初始化后的博客文件夹，解压后可按需重命名使用。

  1. 初始化完毕后，便可以看到blog文件夹里，已经有博客框架的相关文件了。
    （看看它出生的样子…）
  2. 输入hexo s，打开本地服务器端口。
  1. 可以看到端口号是4000，打开浏览器输入网址http://localhost:4000，你就可以看到它的样子了。

注意这里不要用Ctrl+C复制命令行里的网站，因为Ctrl+C会退出服务器。

创建你的第一篇博客

在本地浏览的时候，你会发现已经有了一篇博客，那是hexo初始化blog时默认生成的。
（可以暂时先不用删除，留作纪念…）

现在让我们开始创建我们的第一篇博客吧。

  1. 关闭上面打开的服务器，即在命令行中按下Ctrl+C。
  2. 输入：hexo n '我的第一篇博客'
    即可在/blog/source/-posts/目录（该目录即博客文章存储目录）下生成一个名为“我的第一篇博客”的md文件。

md文件可以用各种markdown编辑器打开，并撰写文章。
当然也可以用其他任意文本编辑器打开，但那写起来不太方便，不能实时查看效果。

小结

经过上述步骤，我们已经实现了利用hexo搭建博客，并写下第一篇文章。
