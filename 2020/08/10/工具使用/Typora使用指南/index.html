    前言

    内容概览：Typora是一款优秀的Markdown编辑器，有着实时渲染（所写即所见）的特色。本文主要从Typora的快捷键与相关常用配置上进行介绍。

    目的：更加优雅的Markdown书写体验

    本文分为以下部分：

    1. 下载Typora
    2. Typora常用配置
    3. Typora快捷键

    下载Typora

    • 官网下载
    • 蓝奏云链接

    从官网下载可能速度较慢，推荐使用蓝奏云链接下载。关于版本问题，蓝奏云链接不保证为Typora最新版本，但可在下载安装后，按需进行更新。

    Typora常用配置

    主题

    设置图片相对路径

    介绍

    经以下步骤，可在使用Typora编写时能实时看到图片，且本地使用 hexo server浏览时也能看到图片。

    无需使用图床作为图片的保存方式，且在浏览时无需使用网络。

    步骤

    1. 通过 Ctrl + ，或菜单进入Typora的偏好设置，点击“图像”设置

    2. 参考图片设置为“复制到制定路径”与路径./${filename}/，即拷贝到与Markdown文件同名文件夹下。

    3. 修改Hexo的全局配置文件_config.yml

    4. 使用 hexo new "博客名"创建一篇博客

    5. 通过Git命令 npm install hexo-image-link --save安装插件

    6. 使用 hexo s 或 hexo server -debug启动服务预览效果

    文件

    • 新建：Ctrl + N
    • 新建窗口：Ctrl + Shift + N
    • 打开：Ctrl + O
    • 快速打开：Ctrl + P
    • 保存：Ctrl + S
    • 另存为：Ctrl + Shift + S
    • 偏好：Ctrl + ,
    • 关闭：Ctrl + W

    视图

    • 显示隐藏侧边栏：Ctrl + Shift + L
    • 源代码模式：Ctrl + /
    • 专注模式：F8
    • 打字机模式：F9
    • 切换全屏：F11
    • 实际大小：Ctrl + Shift + 0
    • 放大：Ctrl + Shift + =
    • 缩小：Ctrl + Shift + -
    • 应用内窗口切换：Ctrl + Tab
    • 打开DevTools：Shift + F12

    段落

    • 标题：Ctrl + 1/2/3/4/5
    • 段落：Ctrl + 0
    • 增大标题级别：Ctrl + =
    • 减少标题级别：Ctrl + -
    • 有序列表：Ctrl + Shift + [
    • 无序列表：Ctrl + Shift + ]
    • 任务列表：- + 空 + [ + 空 + ] + 空
    • 增加缩进：Ctrl + ]
    • 减少缩进：Ctrl + [

    引用

    • 目录：[ + TOC + ] + Enter

    无法在Hexo博客上显示

    • 软换行：Shift + Enter
      硬换行：空 + 空 + Shift + Enter
      换段：Enter

    • 代码：Ctrl + Shift + `

    • 脚注：[^ 1 ]

    • 插入链接：Ctrl + k

    • 锚点：[ + 内容 + ] + ( + # + 标题名 + )

    引用：Ctrl + Shift + q

    $$
    数学符号：Ctrl + Shift + M
    $$

    制表
    Ctrl + T
    • 插入图片：直接拖动到指定位置即可或者Ctrl + Shift + i
    • 分割线：：— + 回车

    编辑

    • 粘贴为纯文本：Ctrl + Shift + V
    • 加粗：Ctrl + B
    • 倾斜：Ctrl + i
    • 下划线：Ctrl + U
    • 删除线：Alt + Shift + 5

    检索

    • 选中一整行：Ctrl + l

    • 选中单词：Ctrl + d

    • 选中相同格式的文字：Ctrl + e

    • 跳转到文章开头：Ctrl + home

    • 跳转到文章结尾：Ctrl + end

    • 搜索：Ctrl + f

    • 替换：Ctrl + h

    [^ 1 ]: [ + ^ + 空 + 数字 + 空 + ]
